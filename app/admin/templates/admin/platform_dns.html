Platform Configuration

Configure the service defaults and MX servers shared by all self-hosted domains.

These settings define the mail infrastructure for all domains marked as self-hosted. New domains added via the provisioning API will inherit these service settings automatically.

Default IMAP Settings

IMAP server hostname and port used for new self-hosted domains.

Default SMTP Settings

SMTP server hostname and port used for new self-hosted domains.

Default CardDAV Settings

CardDAV server for contacts. Leave empty to disable contacts for new domains.

Default CalDAV Settings

CalDAV server for calendar. Leave empty to disable calendar for new domains.

MX Servers

Add one hostname per line. Lower priority numbers indicate primary servers. Each entry must resolve to a valid A or AAAA record and accept connections on port 25.
